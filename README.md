# RECORE コミュニティイベント レポートテンプレート

RECOREコミュニティイベントのレポートを簡単に作成できるテンプレートです。

## 使用方法

### ステップ1: Notionで情報を入力

以下のNotionデータベースを開いて、イベント情報を入力してください：

**Notionテンプレートデータベース**
https://app.notion.com/p/882aa9e9e93041f4be8a65bdd30e702e

1. 「新規」ボタンをクリック
2. イベント名を入力（例：「第1回関西コミュニティ」）
3. テンプレートが自動適用されます
4. 各【編集】部分に情報を入力してください

### ステップ2: 入力項目

Notionテンプレートで入力する項目：

#### 基本情報
- **開催日時**: 例）2026年6月5日（金）17:00～19:00
- **開催場所**: 例）NEXT51 三国ヶ丘店
- **参加者数**: 数字のみ（例）6
- **参加企業数**: 数字のみ（例）3
- **主催者**: 例）RECORE関西コミュニティ

#### イベント概要
- **イベント目的**: イベント開催の背景と目的を記入
- **イベントこだわり**: 開催にあたってのこだわり・ポイントを記入

#### セッション（最大4セッション）
各セッションについて：
- **タイトル**: セッションのタイトル
- **説明**: セッションの詳細説明
- **開催時間**: セッション開始時刻（例）17:00
- **動画ファイル名**: 動画ファイル名（例）session-1700.mp4

#### 参加者の声（4人分）
各参加者について：
- **名前**: 参加者の名前
- **企業**: 参加者の企業名
- **コメント**: 参加者のコメント・感想

#### その他
- **タイムテーブル時間**: 全体の開催時間（例）17:00～19:00（約2時間）
- **成果と示唆**: イベント開催を通じて得たことや今後への示唆
- **メディア**: セッション動画ファイル名、ギャラリー画像ファイル名

### ステップ3: HTMLレポートを生成

Notionで情報を入力したら、以下の方法でHTMLレポートを生成します：

#### 方法1: Claude Codeで自動生成（推奨）
1. Notionテンプレートページをコピー
2. Claude Codeで「Notionデータからレポート生成」スキルを実行
3. HTMLファイルが自動生成されます

#### 方法2: 手動で編集
1. `report-template.html` をテキストエディタで開く
2. 【編集】の部分をNotionで入力した内容に置き換える
3. ファイル名を `report-[イベント名].html` に変更して保存

### ステップ4: ファイル管理

#### 必要なファイル構成
```
report-templates/
├── report-template.html    # テンプレートファイル
├── images/                 # 画像・動画フォルダ
│   ├── about-image.jpg     # イベント概要用画像
│   ├── gallery-img1.jpg    # ギャラリー画像1
│   ├── gallery-img2.jpg    # ギャラリー画像2
│   ├── gallery-img3.jpg    # ギャラリー画像3
│   ├── gallery-img4.jpg    # ギャラリー画像4
│   ├── session-1700.mp4    # セッション1動画
│   └── session-1800.mp4    # セッション2動画
├── report/
│   └── [イベント名]/
│       ├── report.html     # 生成されたHTMLファイル
│       └── images/         # イベント固有の画像・動画
└── README.md              # このファイル
```

## デザイン仕様

- **フォント**: Zen Kaku Gothic New（見出し）、Noto Sans JP（本文）
- **カラースキーム**: グリーン＆ゴールド
- **レスポンシブ対応**: モバイル・タブレット・デスクトップ対応
- **アニメーション**: Fade-in効果（スクロール時）

## トラブルシューティング

### 画像が表示されない場合
- ファイルパスを確認してください
- 画像ファイルが `images/` フォルダに保存されているか確認
- ファイル名が正確に【編集】部分に入力されているか確認

### 動画が再生できない場合
- 動画形式を確認してください（MP4推奨）
- ブラウザが動画形式をサポートしているか確認
- ファイルが `images/` フォルダに保存されているか確認

### レイアウトが崩れている場合
- ブラウザを再読み込みしてください（Ctrl+F5）
- テキストが長すぎないか確認（特に見出しやタイトル）
- 異なるブラウザで試してください
